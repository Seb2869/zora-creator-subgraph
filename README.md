# NFT Editions Subgraph

## Installation

```sh
yarn global add @graphprotocol/graph-cli
```

```sh
graph codegen
```

```sh
graph build
```

```sh
graph auth  --studio <deploy key>
```

```sh
graph deploy  --studio <subgraph slug>
```
